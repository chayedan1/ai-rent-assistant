## 二、依赖安装

```bash
pip install streamlit requests python-docx
```

## 三、Ollama 配置

### 1. 安装 Ollama
```bash
# Windows (PowerShell)
winget install ollama

# 或手动下载
# https://ollama.ai/download
```

### 2. 拉取模型
```bash
# 拉取 Qwen3.5:9b 模型
ollama pull qwen3.5:9b
```

### 3. 启动 Ollama 服务
```bash
ollama serve
```

### 4. 验证服务
```bash
# 测试模型
ollama run qwen3.5:9b "你好"
```

## 四、启动应用

### 1. 进入项目目录
```bash
cd ai_rent_finder
```

### 2. 启动 Streamlit
```bash
python -m streamlit run app.py
```

### 3. 访问应用
浏览器打开：http://localhost:8501

### 4. 进入 AI 租房对话助手
点击左侧导航栏的 "💬 AI 租房对话助手"

## 五、使用说明

### 输入示例
- `你好` - 系统自我介绍
- `押金不退怎么办` - 押金退还法律指南
- `租房合同注意什么` - 合同签订避坑指南
- `房东未经同意进入房间` - 隐私侵权法律分析
- `二房东跑路` - 二房东纠纷维权指南

### 回答格式
每条回答包含：
1. 【问题类型】 - 自动分类
2. 【法律结论】 - 一句话精准结论
3. 【法律依据】 - 《民法典》具体条文号
4. 【实操步骤】 - 分点可执行操作
5. 【维权路径】 - 协商→调解→投诉→诉讼
6. 【证据留存】 - 该场景需要保留的具体证据
7. 【避坑提醒】 - 针对性风险预警

## 六、技术架构


## 七、系统约束

### 严格遵守规则
1. 只回答租房法律问题
2. 100% 基于本地知识库
3. 禁止编造、禁止幻觉
4. 强制格式输出
5. 非租房问题拒绝回答
6. 无答案时明确告知

### 禁止行为
- 不编造法条
- 不虚构条款
- 不夸大风险
- 不提供具体诉讼方案
- 不承诺胜诉
- 不推荐律师
- 不引导非法途径
- 不回答敏感/违法/越权问题

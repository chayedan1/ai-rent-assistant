# AI 租房助手

基于 Streamlit + Ollama + 高德地图 API 的智能租房辅助工具

## 📋 功能介绍

本项目提供以下功能：

### 1. 📢 AI 租房对话助手
- 基于本地 PDF 知识库的 RAG 检索问答
- 使用 Ollama 本地模型（deepseek-r1:8b）
- 严格遵守法律系统提示词
- 输出格式固定：问题类型、结论、法条、步骤、维权路径、证据留存

### 2. 🏠 房源收藏与管理
- 收藏房源信息
- AI 评分系统
- AI 批量对比分析
- AI 评估报告
- 看房提醒功能

### 3. 🚗 房源通勤地图
- 房源位置可视化
- 可交互高德地图
- 自动加载收藏房源坐标
- 点击可打开高德地图网页版

## 🛠️ 技术栈

- **前端框架**：Streamlit
- **本地 LLM**：Ollama (deepseek-r1:8b)
- **地图 API**：高德地图 API
- **知识库**：本地 PDF 文档
- **数据存储**：JSON 文件

## 📦 安装与部署

### 环境要求

- Python 3.8+
- Ollama 已安装并运行

### 安装步骤

1. 克隆项目或进入项目目录

2. 安装依赖：
```bash
pip install streamlit requests json
```

3. 确保 Ollama 服务已启动：
```bash
ollama serve
```

4. 确保已下载 deepseek-r1:8b 模型：
```bash
ollama pull deepseek-r1:8b
```

5. 获取高德地图 API Key：
   - 访问 https://lbs.amap.com/
   - 注册账号并创建应用
   - 获取 Web 服务 API Key

6. 配置高德地图 API Key：
   - 打开 `ai_rent_finder/pages/5_🚗_通勤时间计算.py`
   - 找到 `AMAP_KEY` 配置项
   - 替换为您的高德地图 API Key

### 启动服务

```bash
cd ai_rent_finder
streamlit run app.py
```

服务启动后，浏览器会自动打开 http://localhost:8503

## 🔧 使用说明

### AI 租房对话助手

1. 打开页面：点击左侧导航栏 "💬 AI 租房对话助手"
2. 输入问题：在输入框中输入您的问题
3. 点击发送：点击 "发送" 按钮
4. 查看回答：等待 AI 生成回答

### 房源收藏与管理

1. 打开页面：点击左侧导航栏 "🏠 房源收藏与管理"
2. 收藏房源：在首页搜索房源并点击收藏
3. 查看收藏：在本页面查看所有收藏的房源
4. AI 评估：点击 "📊 AI评估" 查看房源评估报告
5. 看房提醒：点击 "⏰ 提醒" 设置看房提醒

### 房源通勤地图

1. 打开页面：点击左侧导航栏 "🚗 房源通勤地图"
2. 查看房源：在 "房源收藏" 区域查看所有收藏房源
3. 查看地图：在 "🗺️ 地图可视化" 区域查看房源位置
4. 交互操作：点击地图可打开高德地图网页版进行详细查询

## 📝 注意事项

1. **Ollama 服务**：确保 Ollama 服务已启动，否则 AI 租房对话助手无法使用
2. **高德地图 API**：确保已配置高德地图 API Key，否则地图功能无法使用
3. **知识库文件**：确保 `source/租房rag.pdf` 文件存在，否则 AI 租房对话助手无法使用
4. **网络连接**：部分功能需要网络连接（如地图 API 调用）

## 🐛 常见问题

### Q1: AI 租房对话助手无法使用？

**A**: 请检查：
- Ollama 服务是否已启动
- deepseek-r1:8b 模型是否已下载
- 知识库文件是否存在

### Q2: 地图功能无法使用？

**A**: 请检查：
- 高德地图 API Key 是否已配置
- API Key 是否有效
- 网络连接是否正常

### Q3: 收藏房源数据丢失？

**A**: 收藏房源数据保存在 `data/favorites.json` 文件中，请勿删除该文件
